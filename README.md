# Homebrew Tap for Discrakt

This is the official Homebrew tap for [Discrakt](https://github.com/afonsojramos/discrakt), a tool that bridges your Trakt.tv watching status to Discord Rich Presence.

## Installation

```bash
brew tap afonsojramos/discrakt
brew install discrakt
```

## Updating

```bash
brew update
brew upgrade discrakt
```

## Requirements

- macOS 10.13 or later
- A Trakt.tv account
- Discord running on the same machine

## First Run

On first run, Discrakt opens a setup wizard in your browser. Just enter your Trakt username and you're ready to go!

## Supported Architectures

- Apple Silicon (arm64)
- Intel (x86_64)
